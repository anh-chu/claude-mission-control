# Web Research

When performing research:
1. Search the web for the most current information
2. Cross-reference multiple sources for accuracy
3. Focus on practical, actionable findings
4. Save findings to the research/ directory as markdown

## Output Format
- Executive Summary (3-5 sentences)
- Key Findings (bulleted list)
- Opportunities / Risks
- Recommended Next Steps
- Sources (links and references)
